Descrição: O método forEach é usado para iterar sobre cada elemento de um array e executar uma função de retorno de chamada em cada elemento. Uso: É útil quando você deseja realizar uma operação em cada item de um array, como exibir os elementos, atualizar seus valores ou realizar ações semelhantes.

Descrição: O método map cria um novo array, aplicando uma função de retorno de chamada a cada elemento do array original. Ele não modifica o array original. Uso: Útil quando você deseja transformar os elementos de um array em um novo array com base em alguma regra.

Descrição: O método reduce é usado para reduzir os elementos de um array a um único valor. Ele acumula os resultados da função de retorno de chamada em um acumulador. Uso: Útil quando você precisa calcular um valor agregado com base nos elementos do array, como somar todos os elementos ou encontrar um valor máximo.

Descrição:: O método filter cria um novo array contendo apenas os elementos do array original que atendem a um determinado critério especificado em uma função de retorno de chamada. Uso: É usado quando você deseja filtrar elementos com base em uma condição específica, criando um novo array com os elementos que atendem a essa condição.
